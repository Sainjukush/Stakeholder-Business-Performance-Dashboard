# Stakeholder Insights & Performance Tableau Dashboard

This repository presents an interactive Tableau dashboard designed to transform raw insurance stakeholder data into meaningful business insights. The dashboard consolidates premium collection, value creation, profitability, stakeholder demographics, and financial distribution into an accessible and dynamic visual analytics tool.

##  Project Objective

Insurance businesses interact with a diverse range of stakeholders—investors, brokers, customers, and distributors, to name a few. This dashboard was built to empower decision-makers with a holistic view of stakeholder performance, revealing trends, strengths, and opportunities at a glance.

Whether you're a business analyst, executive, or data enthusiast, this dashboard offers an intuitive way to monitor key performance metrics and drive strategic decisions.

---

##  Key Questions Answered

- **Which stakeholder segments create the most value for the business?**
- **How do profitability and premium collection vary across cities, brokers, and tenure?**
- **What are the prevailing shareholder patterns (e.g., promoters, FII, DII, retail) in the insurance ecosystem?**
- **How do cost and commission ratios impact overall profitability?**
- **What are the demographic profiles (age, gender, education, policy tenure) of our stakeholders?**
- **How has premium collection evolved year-over-year?**

---

##  Features & Insights

- **Comprehensive KPIs:** Top-level metrics for premium collected, average stakeholder age, total equity value, and highest value created.
- **Dynamic Filters:** Instantly filter by premium type, city, date, profitability, gender, assets under management, or stakeholder group to focus your analysis.
- **Profitability and Cost Ratios:** Visual breakdowns of net worth, profit after tax, commission, opex, and cost ratios to identify efficiency and margin opportunities.
- **Trend Analysis:** Track yearly premium collection to uncover growth patterns and anomalies.
- **Value Attribution:** See how value created and assets under management are distributed across different insurer categories.
- **Demographic Intelligence:** View stakeholder age, gender, education, and policy tenure distributions to spot dominant profiles.
- **Shareholding Pattern Visualization:** Instantly grasp the balance between promoters, FII, DII, and retail investors.
- **City & Broker Performance:** Identify which regions and intermediaries yield the best returns for various policy tenures.

##  Data Structure and Security

Data covers fields such as stakeholder type, insurer, value created, shareholding patterns, city, brokers, premium types, profitability metrics, tenure, demographics, and asset management.  
**Note:** The sample data provided is anonymized for sharing/public demonstration. When using the dashboard with real business data, ensure compliance with data privacy policies.

---

##  How to Use

1. **Clone** or download this repository.
2. Open `Dashboard.twbx` with Tableau Desktop or Tableau Public.
3. Optionally replace `data/stakeholder_sample.csv` with your actual data (maintaining column structure).
4. Explore, filter, and export insights as needed.
5. Screenshot (`assets/dashboard-preview.png`) provided for quick preview and readme illustration.
